The ChatGPT model (gpt-35-turbo) is a language model designed for conversational

interfaces and the model behaves differently than previous GPT-3 models. Previous

models were text-in and text-out, meaning they accepted a prompt string and returned

a completion to append to the prompt. However, the ChatGPT model is conversation-in

and message-out. The model expects a prompt string formatted in a specific chat-like

transcript format, and returns a completion that represents a model-written message in

the chat.
