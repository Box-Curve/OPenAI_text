If you're ready to train your model, select Save and close to start the fine-tune job and

return to the Models page.

The Models page displays information about your customized model in the Customized

models tab, as shown in the following picture. The tab includes information about the

status and job ID of the fine-tune job for your customized model. When the job is

completed, the file ID of the result file is also displayed.
