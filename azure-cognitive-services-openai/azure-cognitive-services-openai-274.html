Package: com.azure.ai.openai

Maven Artifact: com.azure:azure-ai-openai:1.0.0-beta.1

Package containing the classes for OpenAI. Azure OpenAI APIs for completions and

search.

OpenAIAsyncClient

Initializes a new instance of the asynchronous OpenAIClient type.

OpenAIClient

Initializes a new instance of the synchronous OpenAIClient type.

OpenAIClientBuilder

A builder for creating a new instance of the OpenAIClient type.

OpenAIServiceVersion

Service version of OpenAIClient.
